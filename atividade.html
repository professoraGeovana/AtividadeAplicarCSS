Resumo de 2 livros sobre Webdesign

"Don't Make Me Think" por Steve Krug:

Este livro é um guia prático sobre usabilidade na web. Steve Krug enfatiza a importância de criar sites que sejam intuitivos e fáceis de usar para os usuários. Ele defende a ideia de que um bom design de interface deve ser autoexplicativo, permitindo que os usuários naveguem pelo site sem precisar pensar demais. Krug oferece insights valiosos sobre como simplificar o processo de design e teste de usabilidade, destacando a importância de criar layouts claros, navegação consistente e etiquetas de texto descritivas. Ele também enfatiza a necessidade de realizar testes de usabilidade regulares para identificar e corrigir problemas de design.

"Responsive Web Design" por Ethan Marcotte:

Neste livro, Ethan Marcotte introduz o conceito de design responsivo, que é uma abordagem para criar sites que se adaptam dinamicamente a diferentes dispositivos e tamanhos de tela. Marcotte explora os princípios fundamentais do design responsivo, incluindo grids fluidos, media queries e flexbox. Ele destaca a importância de criar layouts flexíveis e fluidos que possam se ajustar automaticamente ao tamanho da tela do dispositivo do usuário, proporcionando uma experiência consistente em dispositivos desktops, tablets e smartphones. Marcotte também discute as melhores práticas para otimizar imagens e conteúdo para diferentes dispositivos, enfatizando a importância de uma abordagem centrada no usuário para o design web moderno.
